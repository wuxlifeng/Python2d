## 变量 运算与表达式

```python
# Some new support computing type:  
# and or not in is < <= != == | ^ & << + - * / % ~ **
print ('Please input a number:')
number = int(input())
number += 1  
print (number ** 2) # ** means ^
print (number and 1 )
print (number or 1  )
print (not number )
5/2 # is 2  
5.0/2 # is 2.5, should be noted 
```
需要注意的一个是：5/2 等于2，5.0/2才等于2.5。
